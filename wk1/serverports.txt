1. A server port is a numeric identifier in computer networking used for differentiating various services on a computer or server.
2. The server port range is from 0 to 65535.
3. The port number range requires 16 bits for representation.
4. Ten common server ports are:
	HTTP (Port 80)
	HTTPS (Port 443)
	FTP (Port 21)
	SSH (Port 22)
	Telnet (Port 23)
	SMTP (Port 25)
	DNS (Port 53)
	DHCP (Port 67/68)
	POP3 (Port 110)
	IMAP (Port 143)
